# 📖 Quran Companion

Quran Companion is a web application designed to support Quran students in their memorization and revision journey. The project aims to provide simple tools that help users stay consistent, organized, and motivated while learning and reviewing the Quran.

## ✨ Features

* Track memorization progress
* Monitor daily and weekly revision
* Set personal memorization goals
* Organize Quran study routines
* User-friendly and responsive interface

## 🎯 Project Goal

The goal of Quran Companion is to make Quran memorization and revision easier by providing a digital companion that helps students stay focused and consistent in their Hifz journey.

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🚧 Project Status

This project is currently under development and will continue to receive new features and improvements.
